Flagship Program

What the organization is actually building

Winux PTree

Shipping

Filesystem indexer with tree-style CLI integration.

Presented as the fastest open-source Windows indexer tool, with reported indexing speeds below 100 microseconds.

Winux Termix

In development

Hackable terminal environment with fakeroot abilities plus Lua, WASM, and Rust bindings.

Tauri-backed browser virtualization and CSS-driven environments are core to the vision, but the product is still early.

Winux One Bit LLM

Coming soon

Codename Unum. Non-ternary single-bit LLM work with layer-based arithmetic semantics.

Framed as a market-grade custom inference and training engine built for ROCm.
