About

The Winux Foundation exists to end platform tribalism at the user-space layer. We build practical, openly-developed tooling that respects both Windows and Linux workflows without forcing either side to compromise how they work.

Our operating model is simple: publish source, keep docs close to code, prioritize deterministic builds, and ship utilities that solve real operator pain.

Vision

We are currently focused on tools, moving into dev utilities, then a Linux-based OS track. The long-term goal is an XDG, Unix, Linux, and NT32 compliant kernel that intertwines syscall models from both sides without relying on a translation layer.

  • Now: cross-platform tools with clean operator ergonomics
  • Next: deeper developer and systems utilities
  • End state: unified compliance-driven kernel architecture
